# BUG-006: Modal de Tags aparece atrás do conteúdo da página

## Descrição do problema

No módulo de Tags, ao clicar em "Nova tag" ou "Editar", o modal abre mas fica **sobreposto pelo conteúdo da página**.

- O overlay escurece o fundo corretamente
- Mas a tabela, botões e elementos da página ficam visíveis por cima do modal
- O usuário não consegue interagir com os campos do formulário
- O botão "Fechar" fica inacessível

## Arquivo afetado

`modulos/tags/index.php` — CSS da classe `.modal-overlay`

## Resultado esperado

- O modal deve aparecer **acima** de todo o conteúdo da página
- O overlay deve cobrir completamente a tela
- O formulário deve ficar visível, clicável e funcional
- O usuário deve conseguir preencher os campos e salvar normalmente
