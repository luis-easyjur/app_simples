# BUG-012: Exportação CSV omite sempre o último registro de pessoas e agendas

## Descrição do problema

Ao exportar dados pelo módulo de exportação, o arquivo CSV gerado **sempre omite o último registro** cadastrado, tanto para pessoas quanto para agendas. O download ocorre normalmente, sem erros, e o arquivo abre sem problemas — mas os dados estão incompletos.

- Exportar com 1 registro gera um CSV apenas com o cabeçalho (0 registros)
- Exportar com 3 pessoas gera um CSV com 2 pessoas (a última é omitida)
- Exportar com 5 agendas gera um CSV com 4 agendas (a última é omitida)
- Nenhum erro é exibido; o download parece bem-sucedido
- O problema afeta tanto a exportação de pessoas quanto a de agendas

## Resultado esperado

- O CSV exportado deve conter **todos** os registros cadastrados
- A contagem de linhas no arquivo deve ser igual à contagem de registros no sistema
- A exportação deve ser fiel ao estado atual dos dados
