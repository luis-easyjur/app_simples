# BUG-005: Data do lembrete exibida como "Invalid Date"

## Descrição do problema

Na listagem do módulo de Lembretes, a coluna "Data lembrete" exibe **"Invalid Date"** para todos os registros.

- A data é salva corretamente no JSON no formato `YYYY-MM-DD HH:MM:SS`
- Ao renderizar a tabela, o valor formatado aparece como `"Invalid Date"` em vez da data legível
- O problema ocorre em todos os lembretes listados

## Arquivo afetado

`modulos/lembretes/index.php` — função JavaScript `formatarDataHora`

## Resultado esperado

- A data deve ser exibida no formato brasileiro: `DD/MM/AAAA HH:MM`
- Exemplo: `2026-04-25 08:00:00` deve aparecer como `25/04/2026, 08:00`
