# BUG-003: Problemas de interface nos módulos Pessoas e Agendas

## Descrição do problema

Dois problemas distintos afetam a usabilidade dos módulos:

### 3.1 – Módulo Pessoas
- A listagem de pessoas **não carrega**
- Aparece mensagem de erro ou tabela vazia
- As requisições para a API retornam 404 (Not Found)

### 3.2 – Módulo Agendas
- Ao clicar em "Nova Agenda" ou "Editar", o modal abre mas fica **atrás** do conteúdo da página
- A tabela, botões e outros elementos ficam sobrepostos ao modal
- O usuário não consegue interagir com os campos do formulário ou fechar o modal corretamente

## Resultado esperado

### 3.1 – Módulo Pessoas
- A listagem deve carregar as pessoas cadastradas normalmente
- As requisições à API devem retornar 200 com os dados
- O CRUD de pessoas deve funcionar (listar, criar, editar, excluir)

### 3.2 – Módulo Agendas
- O modal deve aparecer **acima** de todo o conteúdo da página
- O overlay deve cobrir a tela e o formulário deve ficar visível e clicável
- O usuário deve conseguir preencher os campos e fechar o modal normalmente
